# android / lineage os

## adb

- [Install adb](https://wiki.archlinux.org/index.php/Android_Debug_Bridge#Installation)

```sh
# reboot
adb reboot
adb reboot recovery
adb reboot sideload

# files
adb pull /sdcard . # backup
adb push FILE /sdcard/
```
